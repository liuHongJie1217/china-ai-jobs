# 中国职业 AI 冲击图谱

中国经济中的每一个职业，在 AI 面前有多脆弱？

本项目基于《中华人民共和国职业分类大典》(2022版)，使用 Claude (Anthropic) 对 **394 个职业**进行 AI 冲击指数评分(0–10)，并以交互式 Treemap 可视化呈现——**面积 = 就业人数，颜色 = AI 冲击程度**。

**[在线体验 →](https://liuhongjie1217.github.io/china-ai-jobs/)** · 灵感来源：[@karpathy/jobs](https://github.com/karpathy/jobs)

---

## 项目内容

- **394 个职业**，覆盖职业分类大典 8 个大类
- 每个职业包含：AI冲击评分、替代类型、时间窗口、中国特有因素分析、评分理由
- **交互式 Treemap** 可视化 + 薪资-冲击散点图 + 学历-冲击箱线图 + 大类排名表
- 支持职业搜索

### 深度洞察

1. **办事人员比工厂工人更危险**：办事人员大类加权冲击指数 6.37，远超生产制造人员的 4.29。白领自动化的威胁大于蓝领自动化。

2. **软件工程师是最大的高危群体**：评分 8/10，1008 万从业者，1-3 年时间窗口。这个中国 IT 行业的核心岗位面临 AI 编程工具的强烈冲击。

3. **学历越高越危险**：学历与 AI 冲击正相关——硕士和本科岗位平均 6.0，初中以下仅 3.82。知识工作比体力劳动更容易被 AI 自动化。

4. **薪资-冲击弱正相关 (r=0.29)**：高薪知识岗位（区块链工程师 22K、投资分析人员 20K）和低薪数据录入岗位共处危险区。

5. **7 个职业面临完全替代**：笔译人员、行政文员、财务出纳、电信业务人员、收银员、呼叫中心服务人员、银行柜员——合计 1857 万人。

6. **体制内岗位的"制度护城河"**：党政机关负责人评分仅 3.49，不是因为 AI 无法辅助其工作，而是制度惯性和政治敏感性构成了额外的替代壁垒。

7. **中式烹饪的文化壁垒**：中式烹调师评分仅 2，涉及手感、火候、味觉判断等难以数字化的技能。

## 数据管线

```
1. 职业编录      → data/occupations.json（职业分类大典 2022，394 个职业）
2. Claude 评分   → data/scores.json（temperature=0，分批处理）
3. 构建站点数据   → site/data.json（压缩格式）
4. D3.js 可视化  → site/index.html（Treemap + 多维图表）
```

## 评分方法论

每个职业基于以下维度评分 (0-10)：

| 维度 | 说明 |
|------|------|
| 任务可自动化程度 | 核心任务中有多大比例可被当前 AI 处理？ |
| 数字化 vs 物理性 | 工作主要是数字化的（高冲击）还是物理性的？ |
| 创造力与判断力 | 是否需要创新性问题解决？ |
| 人际互动需求 | 面对面互动是否不可或缺？ |
| 中国特有因素 | 监管、制度、文化等约束 |
